# Most common registers #
- [a-z] : free to use personal buffers
- [0-9] : last buffers
- / : contains last search query
- . : contains last commadn
- % contains file name
- \+ : clipboard
- \* : primary selection
- = : evaluates expr and prints it
- \_ : the blackhole register
- \- : small delete register (less than a line)
