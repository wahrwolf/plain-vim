# vim has modes #
- command mode
- ex mode
- insert mode
- normal mode
- visual mode

## insert mode ##
- enter stuff
- ctrl+r+register
  * + for clipboard
  * \* for visual
  * = for eval
  * % for file name
  * : for last cmd
  * [0..9] for last yank/delete
- inserts are actions
- substitute, append, insert

## normal mode ##
- work with the file content
- yank/paste
- record macros
- use vim language

## command mode ##
- work with the file
- call functions
- use external tools
- replace stuff

## ex mode ##
- work with vim
- always command mode
- write vim script
